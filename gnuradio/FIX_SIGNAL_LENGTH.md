## 已修复内容

### 1. 添加信号长度常量
```python
SIGNAL_LENGTH = 5000  # 模型期望的信号长度（样本数）
```

所有信号生成函数现在都生成固定长度（5000样本）的信号：
- `generate_clean_signal()` - 无干扰信号

- `generate_chirp_interference()` - 扫频干扰
- `generate_pulse_interference()` - 脉冲干扰
- `generate_noise_interference()` - 宽带噪声

### 3. 修改接收逻辑
接收端现在会收集足够的数据（5000样本）再进行检测。

## 技术细节

### 为什么是5000样本？
这是训练数据的标准长度。查看训练数据：
```bash
python3 -c "import numpy as np; d=np.load('../CCNN/1_datasets/raw/TX_know.npz'); print(d['data'].shape)"
# 输出: (20, 2, 5000)
```

### 信号时长计算
```
时长 = 样本数 / 采样率
     = 5000 / 2e6
     = 2.5 ms
```

这个时长足够短，可以快速测试，又足够长，包含足够的信号特征。
